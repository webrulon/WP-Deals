== Description ==

WP Deals is a special-for-deals plugin where you can make, post, and share deals all around the people just by downloading this plugin and make your own deals-site with it. WP Deals is one of Tokokoo’s network, so after downloading the plugin, then you can explore your site with the nice and gorgeous themes from WP Deals themes.

= FEATURES =

= DEALS =
Quickly and easily create and edit your own deals offer, then assign categories (e.g. software, books, etc.) and tags (e.g. psd, icon, etc.) to your deal offerings.

= ACCOUNTS =
Become our customers, and you will have your own account where you can edit your profile information, view past purchases, and download your vouchers.

= PAYPAL PAYMENT GATEWAY =
We support Paypal payment standard gateway for payment system. You even can add other payment easily base on your preference. More payment gateways will be coming very soon.

= REPORTING AND STATS =
Check the performance of published deals within the deals admin panel to see how many sales you’ve generated, whether the minimum and maximum number of sales were met, and when each deals is expired.

= LIFETIME SUPPORT UPDATES =
Once you become a member, get exclusive access to our customer support forums, feedback forum, documentation, FAQs, tutorials, and articles. As a member, all theme updates and support requests are freely accessible.

= WORDPRESS CORE =
Multiple sidebars from WPDeals themes possibly make all widgets enabled for you. It permits you to quickly and easily customize different areas of your daily deals site. Custom Menus and custom backgrounds are also supported.

= AUTOMATION =
Once you create and launch a deals, everything else is automated. That is, customers pay, the transaction is processed if the minimum number of orders is met, and customers are sent their vouchers/digital files. Each deals automatically expires without any intervention needed and customers will be unable to purchase it thereafter.

= SOCIAL MEDIA INTEGRATION =
Share your site from any social sharing channels (e.g Facebook, Twitter) we’ve been added to the plugin and themes, so you can push more traffic to your site. Then, our Premium Themes also supports Facebook integration to get a quick registrations and logins.

= FULLY CUSTOMIZABLE =
You can use your existing WordPress themes or create your own nice group buying site with our Premium Themes. Upload your own logo, modify deals settings, change color schemes, and alter page layouts to sell daily deals on WPDeals.

= NEWSLETTER INTEGRATION =
By choosing our Premium Themes as your themes, you will be allowed to capture email addresses and deliver daily deals to your subscribers.

= DEDICATED BLOG =
Run your daily deals business and write blog posts from the same WordPress site. This is a terrific way to connect with customers and establish your site’s authority.

= EASY MANAGEMENT =
Super-easy to control your website to WordPress CMS by an integrated plugin and themes from WPDeals.

= MODERN BROWSER SUPPORT =
Our themes and plugin are already tested and approved that they’re compatible with all major web browsers (e.g Internet Explorer, Firefox, Google Chrome, Safari). We also provide you by creating a great aesthetic, taking from some modern browser technologies’ advantages.

= CLEAN EXTENSIBLE CODE =
Well-written and organized code from us make developers easily alter actions without modifying core code. Besides, our themes also support child themes for upgrade compatibility.

== Installation ==

= Minimum Requirements =

* WordPress 3.1 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t even need to leave your web browser. To do an automatic install of WP Deals log in to your WordPress admin panel, navigate to the Plugins menu and click Add New. 

In the search field type "WP Deals" and click Search Plugins. Once you’ve found the plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking Install Now. After clicking that link you will be asked if you’re sure you want to install the plugin. Click yes and WordPress will automatically complete the installation. 

= Manual installation =

If you would prefer to do things manually then follow these instructions:

1. Upload the 'wp-deals' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Deals Settings'
